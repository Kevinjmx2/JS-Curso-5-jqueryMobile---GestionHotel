Realizar una aplicación que permita gestionar los datos de un hotel.
La idea es que la aplicación permita registrar los datos de un hotel, y debe usar al menos los mostrados en las imágenes.

El formulario de registro debe mostrar un mapa con un marcador el cual se podrá arrastrar para determinar la ubicación del hotel en el mapa.

La opción “Lista Hoteles” permitirá listar los hoteles registrados y al hacer clic sobre el hotel, permitirá mostrar una página con los datos del hotel y su respectiva ubicación en el mapa.
